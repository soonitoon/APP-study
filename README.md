# APP

충북대학교 심리학과 파이썬 스터디 소모임 👩‍💻🧑‍💻

## APP이란?

- 뇌-인지공학 융합전공생 및 심리학과 학생들의 파이썬 스터디 모임입니다.
- 파이썬 기초를 함께 공부합니다.
- 조금 더 나은 코드를 작성할 수 있는 방법에 대해 고민합니다.

## 활동 설명

- 일주일 동안 기초 동영상을 보며 공부합니다. 🎬
- 매주 수요일 오후 8시에 화상 회의로 스터디를 진행합니다. 📖
